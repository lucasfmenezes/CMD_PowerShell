# Comandos CMD

Anotações de comandos úteis do CMD para administração, troubleshooting, rede, domínio, GPO, Kerberos e Windows Server.

Mesmo usando bastante PowerShell hoje em dia, o CMD ainda é muito útil em ambientes Windows, principalmente para diagnóstico rápido, servidores antigos, troubleshooting de domínio e situações em que o PowerShell não está disponível ou não é necessário.

## Arquivos deste diretório

~~~text
comandos/cmd/
├── README.md
├── rede.md
├── dominio.md
├── gpo.md
└── kerberos.md
~~~

## Principais temas

- Rede e conectividade.
- DNS.
- Domínio Active Directory.
- GPO.
- Kerberos.
- Logon.
- Replicação.
- Diagnóstico de Windows Server.

## Observação

Alguns comandos precisam ser executados como Administrador para mostrar todos os dados ou aplicar alterações.
