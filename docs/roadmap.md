# Roadmap de estudos

Plano simples para evoluir nos estudos de CMD, PowerShell, Active Directory, Entra ID, Bash e Shell Script.

## Fase 1 - Fundamentos

- Entender terminal, prompt e execução de comandos.
- Praticar navegação em diretórios.
- Criar, mover, copiar e remover arquivos.
- Entender variáveis.
- Entender permissões.
- Entender saída de comandos.

## Fase 2 - CMD

- Rede com `ipconfig`, `ping`, `tracert`, `nslookup`.
- Domínio com `net`, `nltest`, `whoami`.
- GPO com `gpupdate` e `gpresult`.
- Kerberos com `klist` e `setspn`.
- Diagnóstico de AD com `dcdiag` e `repadmin`.

## Fase 3 - PowerShell básico

- Pipeline.
- Objetos.
- Variáveis.
- Condicionais.
- Loops.
- Funções.
- Importação e exportação CSV.
- Tratamento de erro.

## Fase 4 - Active Directory

- Consultar usuários.
- Alterar atributos.
- Gerenciar grupos.
- Gerenciar computadores.
- Gerar relatórios.
- Automatizar tarefas com CSV.

## Fase 5 - Microsoft Entra ID

- Instalar Microsoft Graph PowerShell.
- Conectar no tenant.
- Consultar usuários.
- Consultar grupos.
- Consultar licenças.
- Gerar relatórios.
- Fazer alterações com segurança.

## Fase 6 - Bash

- Criar scripts Linux.
- Trabalhar com arquivos.
- Trabalhar com logs.
- Criar backups simples.
- Testar conectividade.
- Automatizar tarefas com cron.

## Fase 7 - Shell POSIX

- Escrever scripts mais portáveis.
- Evitar dependências exclusivas do Bash.
- Usar `/bin/sh`.
- Criar scripts simples para ambientes Unix/Linux.

## Fase 8 - Automação prática

- Scripts com parâmetros.
- Scripts com logs.
- Scripts com relatórios.
- Scripts com CSV.
- Scripts agendados.
- Scripts versionados no GitHub.
